The **Working Topics** section is located on the left-hand side of the interface. It allows you to create and organize different workspaces based on your projects or tasks.

1. Click the **+** button next to "Working Topics."

## ğŸŒ **Prompt Flows**

Prompt Flows are predefined queries that automate the process of gathering and analyzing data from multiple sources.

### Using Prompt Flows

1. Open the **Prompt Flows** panel.
2. Select a predefined prompt flow (e.g., "Get devices by location").
3. Customize the query if needed.
4. Press **Enter** to execute the prompt flow.

### Creating a New Prompt Flow

1. Click **+ New** in the Prompt Flows panel.
2. Define the query steps.
3. Save the flow for future use.

### Example:

- `@now can you tell me more about device atl01-edge-01?`

## ğŸ¤– **Chat Input**

The Chat Input is where you can communicate directly with your **Tech Buddy** to retrieve data and insights.

### Asking a Question

1. Type your question in the input box.
2. Use the `@` symbol to reference a data source (e.g., `@now`, `@ntb`).
3. Press **Enter** to submit the query.

### Example:

- `@now @ntb Tell me the details of IP address 10.11.128.3`

### Formatting Tips:

- Use `@` to reference a data source.
- Combine multiple sources in a single query.
